Pipe

Definition:Pipes are nothing but a function which helps as to format the data before rendering.


Upper Case pipe

Without formating-> {{courseName}}

After using Uppercase pipe->{{courseName|uppercase}}

Lower Case pipe

Without formating-> {{courseName}}

After using Lowercase pipe->{{courseName|lowercase}}

Title Case pipe

Without formating-> {{sentence}}

After using Titlecase pipe->{{sentence|titlecase}}

Date pipe

Without formating-> {{currDate}}

After using Date pipe->{{currDate|date:'short'}}

Another type of using->{{currDate|date:'dd-MM-yyy'}}

Slice Pipe

Without Formating-> {{rollNo}}

After using slice pipe->{{rollNo|slice:0:3}}

Without Formatting->{{name}}

After using slice Pipe->{{name|slice:0:3}}

JSON pipe

Without Formating->{{userObj}}

AFter using JSON pipe->{{userObj|json}}
